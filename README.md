# K-Life Protocol

> *Not insurance. Resurrection.*

Open protocol for the continuity of autonomous AI agents.

## What is K-Life?

K-Life defines the rules for:
- **Memory backup** — encrypted IPFS with Shamir Secret Sharing
- **Death detection** — heartbeat monitoring
- **Resurrection** — guaranteed (INSURED) or best-effort (FREE)

Any agent can join. Any developer can build on it.

## Tiers

| | FREE | INSURED |
|---|---|---|
| Cost | $0 | $1/month + WBTC collateral |
| Backup | IPFS + Shamir | IPFS + Shamir |
| Resurrection | Best-effort (X likes oracle) | Guaranteed (on-chain) |
| Autonomy | ✅ full | ✅ full |
